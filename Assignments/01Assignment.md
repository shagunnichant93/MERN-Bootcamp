Q2. Who sends the request?

Your Answer:

USER

Almost correct, but not exactly.

The user initiates the action (clicks a button or types a URL).

The browser actually sends the HTTP request.

Example:

You click "Login"

        ↓

Browser sends request

        ↓

Server receives request

Think of it like sending a courier.

You write the letter (User)
The courier company delivers it (Browser)
The company receives it (Server)

So the correct answer is:

Browser
