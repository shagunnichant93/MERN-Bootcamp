## Q5. Why can't we simply type an IP address instead of `google.com` every time?

Your Answer:

> ✅ Easy to remember

**Correct!** 👏

That's the primary reason.

There are a couple of additional reasons you'll learn later:

### 1. Easy for Humans

Which is easier?

```text
google.com
```

or

```text
142.250.182.78
```

Obviously:

```text
google.com
```

---

### 2. IP Addresses Can Change

Imagine Google changes its servers.

Old:

```text
142.250.xxx.xxx
```

New:

```text
172.xxx.xxx.xxx
```

If everyone typed the IP address directly, Google would have to tell billions of users about the new IP.

Instead:

```text
google.com
        ↓
DNS updates automatically
```

Users continue typing:

```text
google.com
```

without knowing anything changed behind the scenes.

This is one of DNS's biggest advantages.
